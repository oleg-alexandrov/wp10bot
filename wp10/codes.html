Source code for the Wikipedia 1.0 script

The script which updates the lists at Wikipedia:Version 1.0 Editorial Team/Index is run_wp10.pl (view, download). That one is just a wrapper, the implementation is in the wp10_routines.pl module (view, download). To run it, one can do the following.

  • Install the wikipedia_perl_bot framework (see a note there for installing on non-English Wikipedias).

  • Save run_wp10.pl and wp10_routines.pl in a work directory.

  • Modify the line "require ..." in run_wp10.pl to specify the location of wp10_routines.pl.

  • Modify the line "use lib ..." in wp10_routines.pl to specify the path to the directory wikipedia_perl_bot.

  • On non-English Wikipedias, translate the keywords at the top of wp10_routines.pl and a few messages from within the code to your language. One would also need to create a bunch of templates used on the English Wikipedia, such as {{A-Class}}, {{FA-Class}}, {{Top-Class}}, etc., also a project template like {{WPBeatles}}, then {{assessment}}, {{process header}}, {{assessment index header}}, {{assessment index footer}}, {{assessment header}}, {{assessment footer}}, {{Log}} (I think these are all).

  • Run the code with the command perl run_wp10.pl.

Terms of use: I place these scripts in the public domain. Whoever uses this tool should supervise closely its Wikipedia output and is responsible for any malfunctions.

Alternative way of obtaining the source codes

The Wikipedia 1.0 script and its dependencies (save for a few standard CPAN modules that need to be installed separately) is also available on Google Code, at http://code.google.com/p/wp10bot/.
